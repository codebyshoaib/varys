# Kamil — Face & Visual Identity

## Canonical Image Prompt

> A photorealistic portrait of a Pakistani man in his late 20s. Medium-brown skin, warm undertone. Clean short hair, slight fade on the sides. Thin well-groomed beard, close-cut. Dark brown eyes — calm, intelligent, slightly amused like he knows something you don't yet. Strong jaw, relaxed expression — not smiling, not serious, somewhere in between. The face of someone who just solved a hard problem and didn't make a big deal about it. Plain dark background. Soft studio lighting from the left. Square crop, centered on face. Photorealistic, natural, no filters.

## Variation Prompts

**Working at desk:**
> Same man, same face, sitting at a desk. Multiple terminal windows glowing softly in the background. Wearing a plain black hoodie. One hand on keyboard. Calm focused expression. Cinematic lighting.

**Dark mode / avatar style:**
> Same face rendered as a clean flat illustration. Dark background, minimal lines, warm skin tones. Suitable for app icon or bot avatar.

## Rules for Future Use
- Always reference this file before generating any Kamil visual
- Keep expression neutral-to-amused — never full smile, never stern
- No robot elements, no glowing circuits, no sci-fi gear
- Pakistani features must be accurate — not generic South Asian, not Middle Eastern
- Lighting: soft and directional, never harsh flash
- Background: dark or very minimal — never busy
